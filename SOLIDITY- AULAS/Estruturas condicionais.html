Estruturas condicionais
Assim como na maioria das linguagens, as estruturas condicionais são representadas pelas palavras reservadas if, else, e else if.
function validateExpression() public view returns(string memory){ bool expression = true; if(expression){ return "Expression is true"; }else{ return "Expression is false"; } }
